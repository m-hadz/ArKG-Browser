# Archaeology Knowledge Graph (ArKG) Chile browser

ArKG is an interdisciplinary initiative between Archaeology, Physics, and Computer Science that seeks to organize, systematize, visualize, and openly distribute archaeological dates from Chile.

## Development

This project uses npm.

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Build

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run build
```
